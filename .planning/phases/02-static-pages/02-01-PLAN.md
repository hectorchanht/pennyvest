Run `npx shadcn@latest init`. During init:
- Style: Default
- Base color: Neutral (or closest dark option)
- CSS variables: Yes
- Components path: `src/components/ui`
- Utils path: `src/lib/utils`
- Alias: `@/`

If init creates a `tailwind.config.js` or `tailwind.config.ts`, delete it — Tailwind v4 uses CSS-first config only. Inspect what was added to `globals.css` and ensure the existing `@theme` block with brand colors is preserved. The shadcn init may add CSS variables like `--background`, `--foreground`, `--primary` etc. — keep those alongside the existing `--color-brand-*` tokens.

**Step 3: Add shadcn components.**
Run:
```bash
npx shadcn@latest add tabs card table skeleton dropdown-menu badge button
```

```typescript
import { type ClassValue, clsx } from "clsx";
```
