1. Create `src/lib/data/yahoo-finance.ts`:
   - `import 'server-only'` at the top (per established boundary pattern)
   - Import `YahooFinance` from `yahoo-finance2` — v3 is INSTANCE-BASED: `const yf = new YahooFinance()` (NOT static import)
   - Import `withCache, TTL` from `./cache`
   - Import `PriceData, EquityPoint` from `@/types/prices`
   - Import `Strategy` from `@/lib/strategies/types`

   Export `getPricesForStrategy(strategy: Strategy)`:
   - Filter allocations where `assetClass !== 'crypto'` (crypto handled by CoinGecko)
   - If no non-crypto tickers, return empty object
   - Cache key: `prices:yahoo:${strategy.slug}`
   - TTL: `TTL.prices` (300s = 5min per D-12)
   - Inside cache fn: call `yf.quote(tickers)` where tickers is an array of ticker strings
   - Map quote results to `Record<string, PriceData>` — for each quote: `{ ticker, price: quote.regularMarketPrice ?? 0, change24h: quote.regularMarketChangePercent ?? null, currency: 'USD' }`
   - Wrap in try/catch — on error, log and re-throw

   Export `getHistoricalForStrategy(strategy: Strategy)`:
   - Cache key: `equity:${strategy.slug}`
   - TTL: `TTL.equity` (86400s = 24hr)
   - Inside cache fn: for each non-crypto allocation, call `yf.historical(ticker, { period1: twoYearsAgo, interval: '1wk' })` where twoYearsAgo is ISO string for 2 years before today
   - Compute weighted portfolio returns: for each week, sum(allocation.weight * weeklyReturn) where weeklyReturn = (adjClose - prevAdjClose) / prevAdjClose
   - Return `EquityPoint[]` with cumulative return starting at 0 (percentage)
   - Handle missing data gracefully — skip weeks where any ticker lacks data

2. Create `src/lib/data/coingecko.ts`:
   - `import 'server-only'` at the top
   - Import `withCache, TTL` from `./cache`
   - Import `PriceData` from `@/types/prices`
   - Import `Strategy` from `@/lib/strategies/types`

   Export `getCryptoPrices(strategy: Strategy)`:
   - Filter allocations where `assetClass === 'crypto'`
   - If no crypto tickers, return empty object
   - Cache key: `prices:coingecko:${strategy.slug}`
   - TTL: `TTL.prices` (300s per D-12)
   - Inside cache fn: call CoinGecko `/coins/markets` endpoint with `vs_currency=usd&ids=${ids.join(',')}&price_change_percentage=24h`
   - Pass `x-cg-demo-api-key` header from `process.env.COINGECKO_API_KEY` if present (per D-11, optional key)
   - Map response to `Record<string, PriceData>` keyed by CoinGecko id: `{ ticker: coin.id, price: coin.current_price, change24h: coin.price_change_percentage_24h, currency: 'USD' }`
   - Use `fetch` with `cache: 'no-store'` (Upstash handles caching)

   Export `getCryptoHistorical(strategy: Strategy)`:
   - Cache key: `equity:crypto:${strategy.slug}`
   - TTL: `TTL.equity`
   - For each crypto ticker, fetch CoinGecko `/coins/{id}/ohlc?vs_currency=usd&days=730` (2 years)
   - CoinGecko OHLC returns `[timestamp, open, high, low, close][]`
   - Downsample to weekly by taking one point per 7-day bucket
   - Return weighted `EquityPoint[]` same format as yahoo-finance historical
  </action>
