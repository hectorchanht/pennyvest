# Phase 02.1: Align Landing Page to Prototype - Context

**Gathered:** 2026-03-27
**Status:** Ready for planning
**Source:** User direction + prototype inspection

<domain>
## Phase Boundary

This phase reshapes the landing page to match the prototype's content-rich, data-driven layout. The prototype shows a **single fund detail page** (Future Tech AI portfolio) with: simulated portfolio chart, holdings table with daily P&L, ranked news with impact analysis, investment opportunities, and portfolio adjustment notes.

Pennyvest's landing page should present a **simplified version** that mixes content from all 4 strategies across 3 investor types, showing news and key data — not just static cards.

**What this phase delivers:**
- A reactive, content-dense landing page that resembles the prototype's information density
- Mixed strategy overview for all 3 investor profiles (conservative/balanced/aggressive)
- Simulated news section with impact analysis previews
- Strategy cards enhanced with mock data (daily change, holdings preview)
- Visual alignment with prototype's styling (dark theme, data tables, impact badges)

**What this phase does NOT deliver:**
- Real API data (Phase 3 handles live prices/news)
- Backend route handlers
- Actual LLM-generated analysis

All data shown will be **static/mock** — the layout and component structure is what matters.
</domain>

## Specific Ideas

### Prototype Reference
The prototype at the Manus URL shows a Future Tech AI fund daily report with:
1. **Header banner**: Fund name + date + "tracking live" badge
2. **Simulated portfolio section**: $1M portfolio, equity curve chart, daily stats (daily gain, cash ratio, 6-month target, annualized return)
3. **Holdings table**: Ticker, allocation %, market value, daily change %, change amount — with green/red coloring
4. **Global News Digest**: 5 ranked news items with impact score (X/10), category badge, source, date, summary, short-term/mid-term impact analysis, related holdings
5. **Investment Opportunity Assessment**: Structured opportunity cards with confidence score, investment logic, catalysts, risk factors, action recommendations
6. **Portfolio Change Notes**: Today's action recommendation, individual holding adjustment details, risk management notes

### Simplified Landing Page Version
For the landing page, create a simplified version:
- Show the profile tabs (conservative/balanced/aggressive) with enhanced strategy cards showing mock P&L data
- Show a "News Highlights" section with 3 mock news items (shorter than prototype's full analysis)
- Each news item: headline, impact badge, 1-line summary, related strategy badges
- Optional: A small "portfolio snapshot" section showing mock total value and daily change per profile
